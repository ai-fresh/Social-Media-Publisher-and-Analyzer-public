Połączone konta

Connect YouTube, Instagram, and TikTok via OAuth 2.0. All connections are managed in the Konta view.

Supported platforms

▶️ YouTube

Upload videos via YouTube Data API v3 with resumable upload.

  • Resumable multipart upload
  • Title, description, tags
  • Custom thumbnail upload

📸 Instagram

Publish Instagram Reels via Instagram Graph API.

  • Instagram Reels (video)
  • Requires Business/Creator account
  • Caption with hashtags

🎵 TikTok

Publish via TikTok Content Posting API with chunked upload.

  • Chunked upload (large files)
  • Title and hashtags
  • Requires TikTok API review

Connecting accounts

In the app, navigate to Konta in the sidebar. Each platform shows its status and a Połącz button. Click it to start the OAuth flow — a browser window opens for authorization via ASWebAuthenticationSession. After approval, the backend receives the callback and stores the encrypted token.

Token security: All OAuth tokens are encrypted with Fernet (AES-256) before being stored in the database. Set ENCRYPTION_KEY in your .env file.

OAuth credentials setup

YouTube

  1. Create a project at Google Cloud Console
  2. Enable YouTube Data API v3
  3. Create OAuth 2.0 credentials (Desktop App type)
  4. Set YOUTUBE_CLIENT_ID, YOUTUBE_CLIENT_SECRET, YOUTUBE_REDIRECT_URI in .env

Instagram

  1. Create a Facebook App at developers.facebook.com
  2. Add Instagram Graph API product
  3. Link a Business or Creator Instagram account
  4. Set INSTAGRAM_APP_ID, INSTAGRAM_APP_SECRET in .env

TikTok

  1. Create an app at developers.tiktok.com
  2. Apply for Content Posting API access
  3. Set TIKTOK_CLIENT_KEY, TIKTOK_CLIENT_SECRET in .env
TikTok production access: The Content Posting API requires a review from TikTok. In sandbox mode, posts go to a test account only.

Scheduling from video detail

Once accounts are connected, open a video, ensure metadata is generated, then click Zaplanuj publikację in the video detail sheet to schedule it for a future date. See the Scheduler page for calendar details.

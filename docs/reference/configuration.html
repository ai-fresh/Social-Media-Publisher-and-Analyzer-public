Configuration

All configuration is done via environment variables in the backend/.env file. Copy .env.example to get started.

Security: Never commit .env to version control. The .gitignore excludes it by default. Especially protect OPENAI_API_KEY, ENCRYPTION_KEY, and all OAuth secrets.

Core settings

VariableDescriptionRequired
DATABASE_URL PostgreSQL connection string. Format: postgresql+asyncpg://user:pass@host:5432/dbname Required
ENCRYPTION_KEY Fernet key for encrypting OAuth tokens. Generate with Python: from cryptography.fernet import Fernet; Fernet.generate_key() Required
BASE_URL Public base URL of the backend (e.g. https://api.yoursite.com). Used for OAuth redirect URIs. Required
ALLOWED_ORIGINS Comma-separated list of allowed CORS origins (e.g. http://localhost:3000) Optional

AI / OpenAI settings

VariableDescriptionRequired
OPENAI_API_KEY OpenAI API key. Used for Whisper (transcription), GPT-4o (metadata), DALL-E 3 (thumbnails), gpt-image-1 (cover editing). Required
GEMINI_API_KEY Google Gemini API key. Used for Imagen 4 thumbnail generation. Requires paid Google AI Studio plan. Optional
Per-project API keys: Users can also set API keys directly in the macOS app (Settings → API Keys). These are stored encrypted in the database and take priority over environment variables. Useful for multi-user setups.

File storage

VariableDescriptionDefault
UPLOAD_DIR Directory for uploaded video files /app/uploads
THUMBNAIL_DIR Directory for generated thumbnail images /app/thumbnails
MAX_UPLOAD_SIZE_MB Maximum upload file size in megabytes 500

YouTube OAuth

VariableDescription
YOUTUBE_CLIENT_IDGoogle OAuth 2.0 client ID
YOUTUBE_CLIENT_SECRETGoogle OAuth 2.0 client secret
YOUTUBE_REDIRECT_URIOAuth callback URL (must match Google Console config)

Instagram OAuth

VariableDescription
INSTAGRAM_APP_IDFacebook App ID with Instagram Graph API
INSTAGRAM_APP_SECRETFacebook App Secret
INSTAGRAM_REDIRECT_URIOAuth callback URL

Full .env example

# Core
DATABASE_URL=postgresql+asyncpg://socialpub:password@db:5432/socialpub
ENCRYPTION_KEY=your-fernet-key-here
BASE_URL=http://localhost:8000
ALLOWED_ORIGINS=http://localhost:3000

# AI
OPENAI_API_KEY=sk-proj-...
GEMINI_API_KEY=AIzaSy...

# Storage
UPLOAD_DIR=/app/uploads
THUMBNAIL_DIR=/app/thumbnails
MAX_UPLOAD_SIZE_MB=500

# YouTube
YOUTUBE_CLIENT_ID=xxxxx.apps.googleusercontent.com
YOUTUBE_CLIENT_SECRET=GOCSPX-...
YOUTUBE_REDIRECT_URI=http://localhost:8000/api/accounts/oauth/youtube/callback

# Instagram
INSTAGRAM_APP_ID=123456789
INSTAGRAM_APP_SECRET=abc123...
INSTAGRAM_REDIRECT_URI=http://localhost:8000/api/accounts/oauth/instagram/callback

# TikTok
TIKTOK_CLIENT_KEY=aw...
TIKTOK_CLIENT_SECRET=abc...
TIKTOK_REDIRECT_URI=http://localhost:8000/api/accounts/oauth/tiktok/callback

Getting Started

Set up the backend, configure API keys, and run the macOS app in under 10 minutes.

Requirements

  • macOS 14 (Sonoma) or later
  • Xcode 15+ (for the SwiftUI app)
  • Docker & Docker Compose (for the backend)
  • OpenAI API key (required for Whisper + GPT-4o)
  • Gemini API key (optional — for Imagen 4 thumbnails)

Step 1: Clone the repository

1

Clone & enter the project

git clone https://github.com/ai-fresh/Social-Media-Publisher-and-Analyzer-priv-.git
cd Social-Media-Publisher-and-Analyzer-priv-

Step 2: Configure the backend

1

Copy the environment template

cd backend
cp .env.example .env
2

Fill in your API keys

Open .env and set the required values:

# Required
DATABASE_URL=postgresql+asyncpg://user:pass@db:5432/socialpub
OPENAI_API_KEY=sk-...
ENCRYPTION_KEY=your-fernet-key-here   # generate with: python -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())"

# Optional — for Imagen 4 thumbnails
GEMINI_API_KEY=AIza...

# Social platforms (fill as needed)
YOUTUBE_CLIENT_ID=...
YOUTUBE_CLIENT_SECRET=...
INSTAGRAM_APP_ID=...
INSTAGRAM_APP_SECRET=...
TIKTOK_CLIENT_KEY=...
TIKTOK_CLIENT_SECRET=...
3

Start the backend with Docker

docker compose up -d

This starts FastAPI (port 8000), PostgreSQL 16, and Nginx reverse proxy.

4

Run database migrations

docker compose exec api alembic upgrade head

Creates all tables: videos, video_metadata, scheduled_posts, social_accounts, brand_settings.

Backend is running! Visit http://localhost:8000/docs to see the interactive FastAPI Swagger UI.

Step 3: Build the macOS app

1

Open in Xcode

Open the SocialMediaPublisher/ folder in Xcode (or double-click Package.swift).

2

Build and run

Select the SocialMediaPublisher scheme and press ⌘R. Alternatively, use the build script:

cd SocialMediaPublisher
bash build_app.sh
3

Connect to the backend

The app connects to http://localhost:8000 by default. If your backend is on a remote server, update the BASE_URL in APIClient.swift.

Step 4: Connect your social accounts

1

Open the Konta view

In the app sidebar, click Konta. The Połączone konta section lists YouTube, Instagram, and TikTok — each showing its connection status and a Połącz button.

2

Authenticate with OAuth

Click Połącz for each platform. A browser window opens for OAuth authorization. Approve access and the token is saved (encrypted with AES-256 Fernet).

3

Configure Ustawienia marki

Still in the Konta view, scroll to the Ustawienia marki section. Choose a thumbnail provider (Imagen 4 or DALL-E 3) and enter a Prompt stylu okładki for consistent thumbnail style.

TikTok Content Posting API TikTok requires approval from TikTok to use the Content Posting API in production. During development you can test with sandbox mode.

Your first video

1

Upload a video

Drag & drop an MP4 file onto the Video Library, or click the upload button. Max file size: 500 MB. Supported: MP4, MOV, AVI, WebM, MPEG, 3GPP.

2

Generate All (one click)

Click Generate All in the video detail panel. This runs transcription → metadata → thumbnail generation sequentially in the background.

3

Review and publish

Review the generated title, description, hashtags, and thumbnail. Select target platforms and click Publish Now or Schedule.

Done! Your video is published (or scheduled) to your connected social media accounts.

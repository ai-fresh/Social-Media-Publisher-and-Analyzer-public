API Reference

Full REST API for Social Media Publisher & Analyzer. Interactive Swagger docs are available at http://localhost:8000/docs when the backend is running.

Base URL: All endpoints are prefixed with http://localhost:8000 in development. Set BASE_URL in your environment for production deployments.

Videos

POST /api/videos/upload Upload a video file

Multipart form upload. Returns the created video object.

Content-Type: multipart/form-data
file: <video file>
GET /api/videos List all videos

Returns all videos ordered by creation date descending.

{ "videos": [...], "total": 12 }
GET /api/videos/{video_id} Get a single video
PATCH /api/videos/{video_id}/metadata Update video metadata 
{
  "title": "string",
  "description": "string",
  "hashtags": ["string"],
  "selected_hashtags": ["string"],
  "captions_srt": "string"
}
DELETE /api/videos/{video_id} Delete a video and its files

AI — Transcription & Metadata

POST /api/videos/{video_id}/transcribe Start Whisper transcription

Runs in background. Sets video status to processing, then ready when complete.

POST /api/videos/{video_id}/generate-metadata Generate title, description, hashtags

Requires transcription to be complete. Runs in background.

POST /api/videos/{video_id}/generate-metadata-with-context Generate metadata with user context

Synchronous. Returns updated video immediately.

{ "user_context": "string" }
POST /api/videos/{video_id}/refine Refine a single metadata field 
{
  "field": "title | description | hashtags",
  "instruction": "string",
  "current_value": "string"
}
POST /api/videos/transcribe-audio Transcribe audio clip to text

For voice input. Multipart form upload, returns plain text.

{ "text": "transcribed text here" }

AI — Cover Generation

POST /api/videos/{video_id}/generate-cover Generate thumbnail (DALL-E 3 or Imagen 4) 
{
  "aspect_ratio": "9:16 | 16:9 | 1:1",
  "user_context": "string"
}
POST /api/videos/{video_id}/edit-cover Edit cover with gpt-image-1

Multipart form with instruction, aspect_ratio, and optional image file.

GET /api/videos/{video_id}/cover-error Get cover generation error 
{ "error": "string | null" }
POST /api/videos/{video_id}/generate-all Run full AI pipeline (transcribe → metadata → cover)

Scheduler

GET /api/schedule List all scheduled posts
POST /api/schedule Create a scheduled post 
{
  "video_id": "uuid",
  "platforms": ["youtube", "instagram", "tiktok"],
  "scheduled_at": "2026-03-15T18:00:00Z"
}
DELETE /api/schedule/{post_id} Cancel a scheduled post

Accounts

GET /api/accounts List connected social accounts
GET /api/accounts/oauth/{platform}/url Get OAuth authorization URL

Platform: youtube, instagram, tiktok

DELETE /api/accounts/{account_id} Disconnect a social account
